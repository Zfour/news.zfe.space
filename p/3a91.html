小爱触屏音箱实现 Stream Deck 贫替

RSSHub

新媒体少数派 sspaiMatrix

发布于：2022年9月18日

次浏览

前几天看到时间线上 latter21 有晒 Elgato 的 Stream Deck，感觉很炫酷自定义程度也很高就想买一个。我一查价格居然要一千多一个，太贵了买不起。然后我又看到「爱运动的数码君」有将小爱触屏音箱安装 Macro Deck 的视频，这就是我想要的：一百多能实现 Stream Deck「贫替」。我折腾完用了几天感觉还不错，今天就写个教程和使用体验给大家分享下。

准备工作

  • 小爱触屏音箱。
  • Windows 电脑
  • Micro USB 数据线
  • 下载需要的文件（包含了驱动、刷机包、搞机工具箱、和安卓端的 App）

安装改版固件

小爱触屏音箱的官方系统是不能安装第三方 app 的，而是需要刷入改版固件才行。安装改版固件不影响小爱同学的正常功能， 也不影响保修。

安装驱动

将文章开头的工具包下载解压，打开 FlashTool - Driver 文件夹里的 Mediatek-Drivers-Install.bat（需要管理员权限），跟随提示完成驱动安装。

进行刷机

打开 FlashTool 文件夹里的 _SPFT.exe 刷机工具，如果弹出报错提示，直接确认忽略即可。

  • 下载 DA 框选择：FlashTool 下 Hikaru_resign-Xiaomi 文件夹里的 DA_Mi_Hikaru-resign.bin
  • 配置文件选择 Image 文件夹里的 MT8167_Android_scatter.txt
  • 验证文件选择 FlashTool 下 Hikaru_resign-Xiaomi 文件夹里的 auth_sv5.auth

然后我们将下方的选框从下载改到固件升级。

点击下载按钮后，通过数据线连接小爱触屏音箱就会开始刷机。刷机的过程可能需要几分钟到十几分钟，在此期间小爱同学触屏音箱会始终处于黑屏状态，只要能在设备管理器中找到其 com 端口即连接正常，不必担心。

等到软件提示完成，将小爱触屏音箱重新接入电源就会开机进入新的系统，第一次进系统会比较慢，耐心等待即可。

Windows | 安装配置 Macro Deck

Macro Deck 是完全免费开源的，可以直接在官网直接下载安装。

将小爱触屏音箱通过数据线连接到电脑，打开搞机工具箱选择软件管理 - 安装软件，选择下载好的 com.suchbyte.macrodeck.apk。已安装的软件可以在小爱触屏音箱改版固件的英语绘本应用里找到。

将 Macro Deck 的电脑端安装完成后，在同一局域网下打开电脑端和移动端 app 即可实现无线连接。Macro Deck 电脑端可以用来配置移动端 app 的功能和界面，基本操作如下：

右键图标可选择编辑，拖动可以进行排序，右下角还可以设置框格的排列。

左侧菜单栏进行可以修改图标、添加显示文字和背景颜色等操作，右边可以设置触发的动作，一般选择 Action。

如果要添加一个桌面程序的快捷方式，在 Windows Utils 选项中添加程序路径即可。

Macro Deck 可以在线下载图标包和插件包， 我安装了以下几个：

  • Windows Utils：支持打开特定的程序文件或一些系统快捷操作
  • Macro Deck Media Controls：支持系统媒体的控制，比如播放暂停下一首
  • SpotifyPlugin：支持 Spotify 的控制，和歌曲信息的显示。（Spotify 需要 Premium 账户，以及在开发控制面板项目添加 http://localhost:5000/callback 的 Redirect URIs）

程序图标也会提示将系统程序图标导入自己的图标包，最后修改下图标就完成了一个按钮。如果觉得图标不够丰富，也可以试试我分享的自用图标。（图标包作者酷安 @paradoxxx）

Macro Deck 也支持一些参数的显示，比如系统自带的 {time} 就能显示时间，一些自带的程序和插件也支持通过这种方式来显示相应的内容。

Mac | 安装配置 Touch Portal

如果你使用的不是 Windows 而是 Mac，那么你可以选择 Touch Portal，也是一款比较有名的媒体控制软件，并且同时支持 Windows 和 Mac。安装逻辑和 Macro Deck 基本一致，这里就不多赘述了。

Touch Portal 的基础功能是免费的，但如果你想设置更多的排列、使用 Spotify、OBS 等这些插件，则需要花十几欧元解锁 Pro 版本。如果只是需要快速启动软件和控制系统功能等快捷操作，免费版已经足够满足需求。

最后

小爱触屏本身也是一直亮屏插电使用的设备，作为 Stream Deck 的「贫替」使用既不影响其原本功能，理论上也不会对使用寿命造成负面影响。我认为这种玩法的实用性是因人而异的，好在 Macro Deck 和 Touch Portal 的可玩性都很高，大家或许可以研究出更高级更有趣的玩法。
