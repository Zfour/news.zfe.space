苹果在新款的 Mac 电脑上推出了 Apple M1/M2 系列自研芯片处理器，出众的性能和优秀功耗使其大获成功！然而苹果芯片采用的 ARM 架构与我们以往常用的 X86 / X64 架构软件并不兼容。

想要在苹果芯片的 Mac 上安装 Windows，除了需要最新版的 Parallels Desktop 虚拟机外，还必须下载 Windows on ARM 版本镜像！目前微软已发布了 Windows 11 ARM 正式版和 Win10 ARM 的 ISO。我们终于能在一些 ARM 设备 / 手机和新款 Mac 上安装流畅运行 Win11 了……

官方 Windows 11 ARM 中文版系统 ISO 镜像 22H2

微软为 Windows 11 ARM 做了非常全面的向下兼容能力，能在 ARM 系统上直接运行传统的 32 位 (x86) 和 64 位 (x64) 的 Windows (.exe) 应用程序。不仅兼容性好，性能上也没明显损失，用户使用时完全无感知，可以直接无障碍地使用以往几乎全部的 Windows 软件 。

也就是说，当你在使用 Win11 ARM 版本系统时，作为用户根本你不必在意所用的软件是何种架构，因为过去所有的 Windows 程序在 ARM 系统下依然可用。

这点与苹果的 M 系列芯片 Mac 通过“Rosetta”转译技术兼容运行过去的 Intel 架构应用一样，Windows ARM 系统同样可以直接向下兼容，而无需任何额外的操作。

目前国内外已有一些玩家将 Windows 11 ARM 系统 安装到树莓派、小米手机等硬件上去跑了，玩法还是不少的。日后估计也还会有更多的创意玩法，比如安装在 ARM 的 NAS 甚至路由器上？不过，目前 Win11 ARM 最广泛的应用还是在 M1/M2 的苹果 Mac 上体现。

M1/M2 系列 Mac 使用 PD 虚拟机安装 Windows 11 ARM

由于目前市场上 ARM 架构的 PC 电脑硬件并不多，比如微软的 Surface Pro X 等，需求最大的反而是采用苹果自研处理器的 Mac 用户！而通过最新版的 Parallels Desktop 虚拟机，我们终于可以在 macOS 上安装和流畅地运行 ARM 版本的 Windows 11 了。

在 PD 虚拟机上运行 Win11 ARM

PD 虚拟机现已支持 M1 直接安装 Win11 ARM 和 Win10 ARM 系统，而 Windows ARM 能兼容运行所有常见的 Windows x86 / x64 (32位 / 64位) 软件，这样一来，新架构的 Mac 在无论是兼容性、实用性都非常完美了，再加上强大的性能和电池续航，真的非常诱人！

而且，随着现在 ARM 的流行，很多新的 Windows 软件在发布时都会同时提供原生 ARM 版本了，性能和兼容性完全不必担心。
