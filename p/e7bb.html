雷军辟谣拒投腾讯阿里：想1000万收购网易，丁磊拒了

雷军在2022年度演讲上表示，“有传言说马化腾创业时要把 QQ 卖给我，马云创业时找我投资我没投。他们两个都没找过我，全是谣言。”雷军说。真正认真谈过的只有网易，当时网易创立只有一年，做个人网站的托管站，他出 1000 万元想收购，但被丁磊拒绝。随后网易融资 1000 万美元，估值 5 个亿，让雷军大为震撼。


被病毒勒索千万美元？美的回应：传闻系谣言，业务系统未受影响

8月11日晚间消息，美的集团在微博上发文回应“美的工厂电脑遭受黑客攻击并勒索千万美元”传闻。美的回复表示，11日当天集团遭受新型网络病毒攻击，少数员工电脑受到感染，公司各业务系统未受影响，经营正常进行，也没有收到勒索信息。据美的内部员工透露，确实有收到部门通知，说不要开电脑，至于是电脑被攻击还是被勒索，该名员工也不清楚。


小鹏汽车回应P7辅助驾驶高架撞人致死，交警：暂不能确认辅助驾驶功能造成

8月11日消息，一辆小鹏P7在8月10日于浙江宁波高架桥上，将一男子撞飞致死的视频引发关注。据网传车主描述，事故发生时车辆开启LCC 2.5辅助驾驶模式，以车速80撞上一辆路边停靠车辆。车主表示“辅助驾驶完全没识别，我撞了后意识到，来不及了。”

小鹏汽车回应称，我们为本次事故中不幸离世的遇难者感到悲痛和惋惜。目前交警部门已经立案处理，门店已第一时间已前往现场协助处理。我们将全力配合相关部门进行事故调查，持续跟进后续结果，并协助客户处理后续相关事宜。随后，负责该案的交警部门表示，目前该事故正进行初步调查，是否为网传的LCC辅助功能造成，其说暂时不能确认。


雷军：小米自动驾驶目标是 2024 年进入第一阵营

雷军在发表 2022 年度公开演讲中透露，小米造车目前进展顺利，甚至可以说超预期。目前正在全国各地进行路测，第一期规划140辆测试车，将陆续在全国进行测试。

雷军表示，小米自动驾驶的目标是 2024 年进入行业第一阵营。在他看来，自动驾驶是智能电动汽车决胜的关键点。雷军进一步指出，小米造车自动驾驶项目第一期研发投入 33 亿元，专属团队规模已超 500 人，到年底有望突破 600 人；并协同人工智能实验室、小爱团队、手机相机等。（雷峰网）


杨元庆回应联想密集招聘汽车研发人才：仍无造车计划

联想集团近期开启了汽车类研发人员招聘。联想集团董事长兼 CEO 杨元庆在业绩会上回应称，联想还没有造车计划，招聘人员是为了做前瞻性研究。（快科技）


马斯克回应套现 465 亿元，或打造新社交平台

马斯克在 8 月 5 日至 9 日总计出售了 792 万股特斯拉股票，价值总计约 68.9 亿美元（约合 465 亿人民币）。对此，马斯克于 8 月 10 日在推特回应表示，出售特斯拉股票是因为他需要现金，以防被迫继续执行收购推特的交易。同时，他还表示，如果收购推特的交易无法达成，他将再次回购特斯拉股票。不过，就在今年 4 月，马斯克就曾在社交媒体公开表示，此后将不会进一步出售特斯拉股票。此外，马斯克在被问及是否考虑自建社交平台时，给出了「X.com」的回答。据悉，马斯克曾在 1999 年创立名为「X.com」的金融服务品牌。该公司于 2000 年被 Confinity 合并，并在 2001 年更名为 PayPal。2017 年，马斯克又从 PayPal 手中购回了「X.com」的域名。（澎湃新闻）


Windows 11 加密 bug 会导致数据丢失运行缓慢

微软证实，Windows 11 存在加密 bug 会导致数据丢失运行缓慢。问题主要影响支持 Vector Advanced Encryption Standard (AES) (VAES) 指令集的最新 Windows 设备。VAES 是 AVX-512 指令集的一部分，英特尔的 Ice Lake、Tiger Lake、Rocket Lake 和 Alder Lake 架构处理器支持 AVX-512。AMD 最新的 Zen 4 处理器也支持 VAES，但它要到秋季才会发布。微软称它在 Windows 加密库 SymCrypt 更新加密指令引入新代码路径时导致了问题。它建议 Windows 11 和 Windows Server 2022 用户安装 2022 年 6 月安全更新，该更新包含了对这一 bug 的初步修复，方法是禁用加密加速。2022 年 7 月安全更新则能将性能恢复到以前的水平。（Solidot）


阿里云盘回应容量全变为100GB：技术故障 已修复

8月10日有大量阿里云盘用户反馈，无论是会员、用户云盘中保存有多少内容，还是用户的云端容量是永久还是临时，阿里云盘的用户容量都变成了100GB。这难道是个BUG？对此，阿里云方面回应称，技术故障导致云盘容量出现异常，目前已修复。


微信支持电脑手机互相迁移聊天记录

据报道，微信 Windows PC 内测版 3.7.6 迎来更新，不过本次更新仅面向已报名用户。新增支持在电脑与手机之间互相迁移聊天记录，迁移后可在手机或电脑上浏览合并后的聊天记录。


新合资公司“Unity 中国”成立：将推出 Unity 编辑器中国版

8 月 9 日消息，游戏引擎开发商 Unity 宣布，Unity 已与一些合作伙伴达成协议，建立一个新的区域性的合资企业——Unity 中国。阿里巴巴、中国移动、吉比特、米哈游、OPPO、佳都科技以及抖音集团等将参与投资该合资公司，Unity 中国的投后估值为 10 亿美元。据介绍，Unity 将把在中国的创建解决方案业务以及游戏服务业务注入新合资公司 Unity 中国。

短期内，Unity 中国将开始为游戏开发者构建其核心产品的本地定制版本，包括旗舰产品 Unity 编辑器的中国版。Unity 中国还将成为 Unity 全球产品和服务在中国地区的独家经销商，确保在中国的客户和创作者社区能够继续访问并使用 Unity 工具。（蓝鲸财经）


AppLovin 开价 175 亿美元收购 Unity

游戏软件公司 AppLovin 向同行 Unity Software 提交收购要约，开价 175.4 亿美元，全部以股票完成交易。之前 Unity 已经宣布要收购 AppLovin 的竞争对手 ironSource，这笔交易可能会告吹。AppLovin 开价每股 58.85 美元收购 Unity，比 Unity 周一的收盘价高出 18%。合并之后 Unity 将会持有新公司流通股的 55%，相当于拥有 49% 的投票权。上个月 Unity 宣称要以 44 亿美元收购 ironSource，随后 AppLovin 聘请顾问提交自己的收购要约。如果 Unity 想与 AppLovin 合并，公司董事会可能会终止 ironSource 交易。（IT之家）


百度无人驾驶网约车起步价 16 元，里程单价2.8元/公里

百度周一表示，已获得重庆和武汉的批准，可以向乘客提供全无人驾驶网约车有偿服务，这些网约车没有司机或安全人员。百度是首家获得此类许可的中国公司。该公司在一份声明中表示，将在武汉指定的13平方公里范围内从上午 9 点到下午 5 点提供这项服务，并在重庆的 30 平方公里范围内从上午 9 点半到下午 4 点半提供此服务。目前百度 " 萝卜快跑 " 已在部分地区推出自动驾驶付费出行服务，以里程和时长作为计费单位，起步价 16 元，里程单价 2.8 元 / 公里。有网友表示，这也太贵了。


员工在下班后用微信工作期间发病，认定工伤

日前，广州铁路运输中级法院二审的判决引发关注。在案件中，广州某贸易公司的员工石某在下班后使用微信继续工作，在工作期间不幸突发疾病，最终身亡，被二审法院认定为工伤。二审法院指出，石某的微信聊天记录显示，其经常下班后用微信回复工作信息。可见，石某回家后继续处理工作是常态且具有连贯性，属于《工伤保险条例》中规定的「工作时间和工作岗位」的延伸。因此，在此期间突发疾病死亡的，应当视同工伤。

对此，深圳市总工会官方发文表示：“微信等具备实时发送文字、语音、图片、视频功能的通讯工具低成本、高效率，职工可以随时随地在线处理工作，一定程度上打破了传统的线下坐班、处理工作的时空限制。因此，对于相关法律法规的理解，也应当立足于当前社会经济发展状况，作出公平公正、兼顾劳动者权益的解释。”（爱范儿）


周鸿祎否认360不能卸载：提供完整卸载过程

新东方创始人俞敏洪对话360集团创始人周鸿祎，俞敏洪问“怎么用360卸载360？”，周鸿祎表示，360自己不能卸载自己的说法是以讹传讹，都是谣言，360提供自己卸载自己的能力。“有些小朋友用Windows随便拖拽，删到垃圾箱肯定不行，我们会区别是用户在卸载还是被攻击。”


推特确认540万账户数据泄露：漏洞已修补，正通知受影响用户

7月22日Restore Privacy报告称，推特因安全漏洞被黑客入侵，共计540万个账户的联系方式泄露，泄露的540万个账户包括推特ID与其关联的电话号码和电子邮件信息，已在一个黑客论坛上出售，价格为3万美元（约20.28万元人民币）。今日，推特已正式确认该攻击已发生，并且该0-day漏洞已被修补。（IT之家）


Argues JSON 作者认为应该退役 JavaScript

Argues JSON 作者 Douglas Crockford 认为最流行的编程语言 JavaScript 如今已成为进步的障碍应将其退役，但他同时承认目前不存在 JS 语言的替代。根据 StackOverflow 的调查，逾 65% 的开发者使用 JavaScript，第二名的 Python 只有 48%。Crockford 认为 JS 的问题是它日益臃肿，社区对增加功能而不是修正其缺陷更感兴趣，JS 和其它恐龙语言一样成为了进步的障碍。Crockford 称取代 JS 有两大难点：第一是还不存在 JS 替代品，第二需要所有浏览器都采用它。结果只能是“祝好运”了。（Solidot）


华为完成首次6GHz频谱试验

近日，阿联酋电信公司与华为一起完成了其首次6GHz频谱试验，这在5G行业诞生以来有着很大的意义，5G技术主要依靠中频频谱来实现安全和长期性能，6GHz 将进一步提高效率。（中关村在线）


Salesforce 中国区宣布解散 业务将由阿里云代销

据牛透社8月3日消息，Salesforce中国区宣布解散，Salesforce位于香港的办公室也被关闭。此前，Salesforce中国区主要负责中国大陆、中国香港和中国台湾业务。目前，中国大陆和中国香港业务将交给阿里云代销。中国台湾由新加坡办公室管理。阿里云与Salesforce的战略合作始于2019年7月25日的云栖大会，双方低调宣布战略合作，阿里巴巴成为Salesforce在中国大陆、香港、澳门、台湾地区的独家客户服务提供商。但据知情人士透露，此前，双方的合作落地较慢。在新冠疫情后，双方合作加速。


Gitlab 禁止使用 Windows

Gitlab 被发现有一项禁止使用 Microsoft Windows 的公司政策。微软旗下的 GitHub 是 Gitlab 最主要的竞争对手。Gitlab 表示它允许员工使用 Linux 和苹果 macOS，而禁止使用 Windows 的理由是它是份额最高的桌面系统，因此也是间谍软件、病毒和勒索软件的最主要目标。Windows 家庭版的安全性难以保障，因此不符合公司安全准则。苹果的电脑是首选，如果熟悉 Linux 也可选择戴尔公司的 Linux 笔记本电脑。（Solidot）


以全栈信创产品，安全赋能智慧能源！普元出席中国核电信息技术高峰论坛

近日，第十届中国核电信息技术高峰论坛（NITF 2022）在浙江海盐成功召开。享有“亚洲首个专注于核电信息技术发展的交流平台”之誉的论坛，现场汇聚了多领域专家深入探讨“智慧核电 助力双碳 赋能未来”。作为值得客户信赖的基础软件技术公司和信创产业链的关键成员，普元受邀出席论坛。普元数智研究院资深顾问左磊，结合企业数字化转型与行业信创发展趋势，发表题为《打造企业数据资源平台 提升企业数字化运营能力》的主题演讲，与现场专家交流普元基于全栈信创中间件产品的领先优势，为客户提供安全可靠的数据管理与治理产品与解决方案，从源头解决信创应用迁移的数据问题，以普元数据治理运营一体化方法赋能核电企业智慧转型的实际应用案例。


携手华为助力客户高效构建高可靠应用，普元出席鲲鹏开发者创享日·长沙站

８月５日，由华为技术有限公司主办，湖南省鲲鹏生态创新中心等协办的“鲲鹏开发者创享日·长沙站”在线上与线下同步举行。在这场由华为推出的面向鲲鹏计算产业全栈开发者的活动现场，汇聚了国内顶尖技术大咖、科研带头人、知名企业技术专家及高校开发者，围绕“创未来，享非凡”主题描绘计算产业发展趋势和蓝图，展现鲲鹏全栈创新技术、商业应用等生态全景。普元作为鲲鹏计算产业生态的重要伙伴，受邀出席活动，与专家们共同交流技术创新，展望鲲鹏计算产业的未来。
