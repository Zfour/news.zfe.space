Swing 是 Java 的图形 widget 工具包。当前的 JetBrains IDE 都使用了 Swing 来绘制 UI，其他基于 IntelliJ 的 IDE （例如 Android Studio）同样如此。尽管 JetBrains IDE 和 Swing 应用都是桌面应用，但存在一些边界情况需要使用特殊方法，比如瘦客户端 (Thin clients)，这就是 Projector 的使用场景。
