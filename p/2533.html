全平台免费好用的 OCR 扫描识别 / 图片转文字工具

白描 APP 支持手机拍照扫描以及 OCR 文字识别 / 图片转文字等功能，它不仅好用，而且平台覆盖还很全面。目前已支持 Android、iOS、微信小程序、网页版等主流系统平台，同时它还支持 Windows 和 macOS 桌面版，支持屏幕截图提取文字，在电脑办公时也能获得极大的便利！

白描的识别效率很准确率都非常高、功能也很全面，使用起来非常顺畅方便。无论是办公、学习、还是日常生活，它都能在各个方面帮你“轻松抓取拿下”现实中你能看得见的所有文字！手机拍照、桌面截图、图片识别样样都行，绝对是人手必备的一款 APP。

白描 Windows 版： 桌面截图识别提取文字：

白描 Windows 版最方便的地方在于支持「截屏识别」，功能上与「天若 OCR」类似。当软件运行后，你只需点右下角托盘的小图标，选择「截屏识别」就可以在屏幕上任意区域划取识别了！所有文字都能信手拈来！

如果你经常需要识别提取屏幕上的文字，也可以为此功能自定义一个快捷键，随时“一键呼出”截屏识别，这取字的效率绝对是飞跃式的提升。

支持表格识别、公式识别、PDF 提取文字

文字识别只是最基础的功能。在实际电脑办公的场景下，我们经常需要处理一些纸质打印文档 / Excel 表格照片或扫描图，甚至是手写的字条。

而白描可以非常好地支持识别表格 (可以导出为 Excel 文件)、数学公式 (可以导出 LaTeX 格式) 以及手写文字、PDF 文档提取文字等。

批量图片转换文字：

如果你有大量的文档照片、扫描图片、PDF 扫描版文档等等，都可以使用白描进行批量的文字提取，只需将图片拖放到白描界面上即可实现一键批量识别并导出，非常高效。

白描手机版 APP：拍照文字识别 / 纸质文档扫描

而使用手机版的白描 (iPhone、iPad、Android) 等，你只需将手机摄像头对准要识别的纸质文档、屏幕、或现实环境中各种的文字，点击拍照即可立即进行「扫描」和「文字识别」了。之后你可以可以选择图片上的文字，并且帮你生成可以编辑的内容了，复制加工转发随你方便，而且还能导出成 TXT、DOCX 等文档格式。

[ 白描-OCR文字识别与文件扫描全能处理 iPhone 版截图 ]

白描可以识别出中文简体与繁体文字、以及英文、德语、法语等语言，同时对手写字的识别也有高准确度。实测白描的识别准确率还是很高的，识别后也能非常方便地校对和修改。

文字翻译

而且白描 APP 还内置了翻译功能，可以方便地一键获得翻译后的结果。翻译功能支持中文、日语、英语、韩语、法语、西班牙语、阿拉伯语、俄语、德语、葡萄牙语、意大利语、繁体中文、粤语、文言文的互译。

扫描纸质文档 / 自动识别文档边界 / 生成高清扫描件 / 身份证扫描

当你拍摄纸质文档时，只需将摄像头对准文档，白描就会自动识别出文档的边界，并自动生成高清晰度的扫描件。软件支持多种色彩的切换，并可快速将多页合成 PDF 文档。

而针对生活中经常需要用到的身份证复印件，白描也支持了「身份证扫描」功能，可以非常方便地生成身份证正反面 A4 扫描件。

表格识别

如果你的工作中经常需要处理一些包含表格数据的纸质文件，需要录入到电脑上去，那么白描的「表格识别功能」绝对是你的救星！因为很多同类的 OCR 扫描应用都是只能识别文字，而不能识别出表格，往往还需要大量繁琐的手动编辑，才能在 Office (Excel)、WPS 上重新排版成表格。

而白描内置了“表格模式”，你只需切换到此模式后进行识别，就能获得包含表格的识别结果，只需核对或轻度修改一下即可生成 Excel 电子表格，绝对的省时省力！比起手动做表格不知道要高效率多少倍！

批量扫描识别文字：

当你需要进行大量的文档扫描识别时，白描的「批量识别文字」功能就很好用了！你只需在 APP 的界面上长按图片即可开启“批量识别模式”，一次可以选择多张图片进行批量识别。

同时，白描也支持批量合并图片并导出 PDF、批量文字识别并合并结果、批量打印等功能，可以大大加快你处理各类文档的效率。

白描网页版：

除了手机 APP 和 Mac、Windows 等电脑版的客户端之外，白描还很贴心地提供了「网页版」。在任何电脑上，你都能直接通过浏览器上传图片 (支持 JPG/PNG/BMP/GIF/SVG 格式) 实现文字识别、数学公式识别、表格识别；同时，它还支持上传 PDF 文档转换文字等；

网页版同样支持上传多张图片批量识别，支持简单的剪裁编辑，以及一键将结果导出成 DOCX、TXT 等格式文件，非常的实用。即使在 Linux 等系统上也可以使用。

白描 macOS 桌面版客户端：

当然，如果你使用 Mac，也可以下载安装白描的 macOS 电脑桌面版客户端，它的功能与 Windows 版基本一致，比如截图识别、批量转文字等，可以大大方便常在电脑前工作的朋友。

[ 白描-OCR文字识别与文件扫描全能处理 Mac 版截图 ]
