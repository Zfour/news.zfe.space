设计性能体验全面提升 vivo X Fold+折叠屏正式发布

RSSHub

新媒体cfan新闻

发布于：2022年9月26日

次浏览

2022年9月26日，vivo正式发布折叠屏手机X Fold系列的高配版本——X Fold+。在vivo X Fold的基础上，vivo X Fold+在设计、性能、体验等方面全面提升，不仅搭载了行业领先的骁龙8+ 定制SPU安全芯片，还进一步增强了产品的续航能力，同时为消费者带来更好的软件体验以及全新的华夏红配色。

vivo产品副总裁黄韬表示：“从 vivo 首款折叠屏 X Fold 开始，我们就将它定位为专业折叠旗舰，这不仅仅是对一款产品的重视，更代表vivo 对折叠形态的未来发展充满信心。”从发布之日起，包括120Hz折叠巨幕、3D超声双屏幕指纹、50W无线闪充等在内的多项折叠屏全球领先技术，让vivo X Fold成为真正意义上的集大成者，并引领折叠屏手机市场进入2.0时代。秉承敢于追求极致，持续创造惊喜的理念，vivo X Fold+不论是硬件性能，还是软件体验都更进一步，持续打造更强的折叠屏旗舰的产品价值和一用难回的使用体验。

国庆特别版上市 vivo X Fold+新增配色华夏红

在vivo X Fold上，vivo提供了晴山蓝和梧桐灰两种配色，一个代表了勇气与希望，另一个则代表了高洁的品质。但无论是哪种配色，灵感和出发点都来自于中国传统文化的经典意象。此次，vivo X Fold+在这两种配色基础上，增加了全新的华夏红配色。对于国人来说，红色是贯穿华夏文明的深厚底色，象征着民族的自信与奋进，流淌在每个中国人的基因里。而此次全新配色华夏红的推出，也代表了vivo 对于华夏文明锐意进取、开拓创新的致敬。

而在设计理念上，vivo X Fold+依然沿袭了前作“天圆地方”的设计语言，将方圆这一具有美好意向及富含哲学底蕴的元素融入到手机产品中。以方形为基础，通过方与圆的结合、阶梯状分布、材质碰撞与纹理修饰等手段，将华夏文明“方”与“圆”的思辨精神，融入到设计表达中。

强者更强 vivo X Fold+搭载骁龙8+定制SPU安全芯片和80W双电池闪充

凭借强大的整体配置，vivo X Fold一经面世就被誉为“折叠屏天花板”。此次，vivo X Fold+在配置方面进一步升级，可谓“强者更强”。处理器方面，vivo与高通合作，联合研发了骁龙8+定制SPU安全芯片。与X Fold的处理器相比，骁龙8+定制SPU安全芯片最大的变化是采用了更先进的4nm工艺制程，性能更强、功耗更低，真正做到了性能与功耗兼顾。

续航方面，vivo的工程师在不增加机身体积和重量的前提下，将X Fold+的电池容量增加到了等效4730mAh*，整体续航提升超过20%。有线充电方面，vivo X Fold+搭载折叠屏首发80W双电池闪充，至快18分钟左右可以充至70%，至快35分钟左右充电至100%。无线充电方面，50W无线充电至快4分钟充至10%，至快53分钟充至100%。最后，vivo X Fold+还标配PD双Type-C插口氮化镓充电器，能够满足多个电子设备的充电需求，日常差旅场景使用起来更加从容。

不仅如此，依托vivo与高通深度联合调校，一方面，骁龙8+定制SPU安全芯片的性能得到全面释放，另一方面，vivo X Fold+在通信、隐私安全等场景中给用户带来独一无二的全新体验。vivo X Fold+是行业内首款双5G双卡双通折叠屏手机，区别于双卡双待单通，这种双5G双卡双通可以实现不断话，不断网，不漏接，有效保障了用户使用双5G卡时，在通话、游戏、使用数据网络等双卡并发场景的顺畅。

另外，vivo X Fold+的这颗定制SPU安全芯片基于CC EAL4+安全等级设计，能够满足金融级场景的安全要求，它拥有独立的安全硬件处理器及独立的安全操作系统，并深入集成到芯片中，赋予产品硬件级的安全防护能力，像日常使用的帐号密码都可以在SPU内安全加密。同时，vivo X Fold+还支持隐匿模式和原子隐私系统，用户数据和使用场景更加安全。

内外皆惊艳 vivo X Fold+搭载内外双旗舰主屏

折叠屏手机一用难回的使用体验是从vivo X Fold+的巨幕开始的，vivo X Fold+配备了8英寸超大内屏，分辨率高达2160*1916，带来可与平板电脑媲美的观看和操作体验，更可让办公场景下的使用、尤其是浏览/编辑文档更便捷。vivo X Fold+的内外屏均支持120Hz高刷新率，内外屏切换过程中视觉感受连贯，内屏和外屏在滑动、游戏或显示高帧率视频时画面更流畅，实际体验更跟手。

作为折叠屏手机最为关键的部分之一，铰链的好坏直接关系到用户体验。vivo X Fold+搭载了航天级无忧铰链，并通过了TÜV折叠无忧认证，在折叠、触控、显示、开合力等关键参数部分表现优异，真正做到了折叠可靠，还可以为用户带来稳定的使用感受。同时，配合锆合金浮动中板设计和UTG超韧玻璃，vivo X Fold+真正做到了折叠无忧、展平如镜。

解锁方式上，vivo X Fold+配备了3D超声双屏幕指纹，这种解锁方式突破了业内千篇一律的侧边解锁，实现了创新性的指纹识别技术突破，抗环境干扰能力更强、解锁速度更快、安全性更高。

拒绝大而无当 vivo X Fold+打造一用难回的使用体验

相对于普通手机来说，折叠屏手机的“大”是先天优势，但如何将这种优势彻底发挥、拒绝大而无当却是摆在手机厂商面前的一个重要问题。从2020年开始，vivo 积极推动成立“金标联盟”，并牵头制定了一整套行业标准，让移动智能终端的生态环境进一步标准化。今年8月，vivo联合众多厂商发布了《大屏适配白皮书》，让庞大的应用开发者能够更高效率、更低成本地加入折叠屏的大生态，为用户带来更好的软件生态和适配。目前，vivo X Fold+已经实现TOP 5000+ APP兼容 、200+ APP重点适配，其中 WPS 、微信读书等高频APP更是实现了深度适配。

vivo X Fold+的超大内屏能够更好的贴合商务办公人士的使用习惯，其内置的应用分屏、应用分身、微信平板模式等可以大幅提升办公效率。不仅如此，vivo X Fold+的超级多任务前台可同时开启最多2个小窗，分屏双小窗可以支持4个任务App同时操作，后台最多可同时挂起12个小窗应用，仅需一部手机就可以同时完成方案编辑、PPT审核、文件传输、微信沟通等一系列操作。

依托于悬停功能，vivo X Fold+可以实现一系列独特的交互体验。在WPS软件上，vivo X Fold+支持演讲者模式，悬停状态下的X Fold+上面是PPT、下面是备注，并且可以通过WiFi直接投屏，使用感受无限趋近与笔记本电脑。悬停通话、悬停观影等功能更是可以帮助用户解放双手；另外，vivo X Fold+的外屏显示锁定支持帐篷模式、屏风模式和台历模式，为用户带来一用难回的使用体验。

蔡司全焦段旗舰四摄 vivo X Fold+打造折叠屏影像天花板

影像部分，vivo X Fold+继续联合蔡司，让手机具备强大的影像性能。vivo X Fold+镜头都搭载了光学性能优秀的蔡司T*镀膜，这种镀膜拥有强大的抗眩光能力，并且可以增强可见光透过率、提高画质，还可以准确还原物体色彩，为高端人群提供优质纯净的光学体验。而为了让vivo X Fold+拍的更全面，更是搭载了旗舰少有的全焦段四摄，真正做到了远近皆好、所见即所得，配合影像算法矩阵，好照片信手拈来。

当蔡司遇上折叠屏，更多的影像新玩法即将解锁。vivo X Fold+支持外屏帮拍功能，内外屏同时显示拍摄画面，用户可以通过外屏预览显示看到自己的画面，调整姿势及表情，进而拍出更加满意的照片。同时，vivo X Fold+还支持后置自拍，解析力、感光力、视野范围等综合素质均优于前置摄像头，自拍效果更加出色。另外，X Fold+还支持悬停自拍、延时摄影，利用手机本身的多角度悬停形态，结合vivo的手势、手掌快门等方式，彻底解放用户双手。

除此之外，vivo在发布会上宣布再度与FIFA世界杯达成合作，vivo X Fold+将成为2022FIFA卡塔尔世界杯TM全球官方手机。致力于为FIFA组委会和世界杯全球球迷带来一届更数字化、更紧密连接的难忘赛事，X Fold+将同vivo其他旗舰产品一起，为FIFA全体组委会及相关团队带来更专业、更可靠的工作支持。

价格方面，vivo X Fold+提供12GB+256GB和12GB+512GB两种配置，价格分别为9999元和10999元。

目前，vivo X Fold+已在vivo官网、vivo京东自营官方旗舰店、京东vivo官方旗舰店、天猫vivo官方旗舰店、vivo手机苏宁自营旗舰店、天猫苏宁易购官方旗舰店、拼多多、分期乐、唯品会、招商银行掌上生活、平安银行口袋商城、vivo体验店专卖店、vivo终端各大门店等线上线下渠道开启预售。首销期间（9月26日-10月7日），预定并购买 vivo X Fold+ 的用户，有机会获得价值528元的限量大礼，包括50W无线闪充立式充电器及素皮支架保护壳。同时，购买X Fold+可以享受最高1700元的以旧换新补贴， vivo老用户可额外获得最高200元补贴；在线上以及线下指定体验店购机，可享受最高6期免息，购机后登录vivo账号还能领取一系列专属购机特权。
