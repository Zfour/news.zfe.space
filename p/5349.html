互联网时代，浏览器成了我们阅读工具流中不可或缺的模块。但在浏览网页时，你一定遇到过这些问题：

  • 在网页上阅读文章的时候，总被「狗皮膏药」般的横幅广告打扰；
  • 文章中想要摘抄的内容，又没法一步到位地管理；
  • 装了一堆插件，却没有提高多少阅读效率；
  • ……

这些令人头疼的事情无不一次次打击着我们想要从网页文章攫取知识的信心。结果，我们做了很多努力，收获的知识却寥寥无几。

简悦则是为你解决这些烦恼的好帮手，经过多年的积累，它俨然从最初的「阅读模式」插件变成了具备完整知识管理能力的生产力工具。它可以开启阅读模式和插件，细致地调整阅读选项，提供了周全的标志工具，甚至还能通过开发接口，和其他知识管理工具进行联动。

然而，这些丰富的功能和可调节选项带来了一个问题：上手门槛高。这是因为知识管理本身就不是一件容易的事，而要熟悉、适应一款工具就更需要额外的时间。

于是，我们为你带来了这套《59 分钟就上手简悦 2.0》付费栏目，帮你将简悦看似繁重的功能拆分成多个板块，细致地分析它的具体用法和实际效果，降低上手门槛，让你的简悦也能像一盒炸鸡那样「开盒即食」。

无论你是知识管理的老手，还是想要尝试简悦的小白，相信这套栏目都能帮你更快上手简悦。

这份栏目如何带你上手简悦

简悦的功能非常丰富，可以满足我们对于网页文章的诸多需求。打个比方，如果你只想让阅读文章的体验变得更加优雅美观，那么简悦的「阅读模式」和各类插件绝对可以让你打造属于自己的「阅读室」。

如果你想高效地将文章里的知识点归纳起来，进行系统性管理。那么简悦的标注、开放平台、稍后读等功能，让它足以成为你知识管理工作流里的一把好手。

我们并不只想让你知道怎样「使用」简悦，而是希望通过这个栏目帮助你找到最适合自己的用法。因此，栏目并不只是简悦 2.0 的简单说明书，而会结合实际用例、需求来解决你在网页浏览、知识管理中可能遇到的问题，并基于简悦 2.0 提供解法，帮你提高阅读体验、知识管理效率。

于是我们推出了新栏目《59 分钟就上手简悦 2.0》，属于「59 分钟就上手」系列，点击了解为什么是 59 分钟。

这份栏目适合谁？

简悦是一款服务于网页阅读的扩展插件，在提供「如杂志般沉浸式阅读体验」的同时，2.0 版本加入的标注、剪藏、稍后读、同步助手等功能，让简悦升级成了读者的「知识管理粘合剂」，帮助读者进一步加工、管理、输出自己阅读的信息。

本栏目在设计之初，就是面向零基础、对简悦不了解的读者，但却不止于此。为了适配不同读者的需求，找到合适的上手路径，本系列教程将分为三级：

  • 轻量级：通过阅读模式和插件提升网页阅读体验。着重介绍阅读模式的基本使用方式，帮助零基础用户快速上手，实现最基础的阅读需求。
  • 中量级：少量配置实现标注和三方应用的自动同步。着重介绍标注支持的功能，并配合笔记工具 Obsidian、Logseq 进行标注自动同步的实际案例，实现对知识的加工和初步管理。
  • 重量级：稍后读结合简悦开放平台，建造你的数字花园。着重介绍稍后读独立界面支持的功能，借助简悦开放平台和 API 工具，导出简悦到任意生产力工具，建立基于双向链接的数字花园，从根本上提高生产力。

跟随本系列内容，你不仅可以快速上手使用简悦，还能厘清不同的阅读模式和多种同步配置，打造属于自己的知识管理工具。如果你已经使用过简悦，却曾在众多 API 玩法中迷失，那么，本系列内容则可以作为你通向更高阶玩法的钥匙。

需要提醒的是，无论从定位还是内容形式， 本栏目都是定位在「入门」级别的栏目，而不是「完全指南」或是「完整教程」。如果有任何疑问，在下单购买前可以私信 @数字工具组，来询问自己是否适合这款栏目。

「我」能从栏目中获得什么？

作为「59 分钟」系列的栏目，我们希望你在阅读完内容之后，能够达成以下效果：

  • 掌握简悦的使用逻辑与操作技巧。能够轻松通过不同类型的阅读模式和标注功能，提升网页文章的阅读体验，并实现对信息的初步加工。
  • 搭建工具间的桥梁。能够通过少量配置与 Obsidian、Logseq 等第三方应用自动同步，实现内容在笔记工具上的实时同步，统一管理。
  • 建造数字花园，完成知识管理。掌握稍后读功能丰富的设置细节，结合简悦开放平台，借助适用于全平台的收藏助手，构建自己的知识网。
