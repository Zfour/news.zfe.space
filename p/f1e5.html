回顾一周社区热门资讯

第【九十七】期：20210130-20210205 新年好

Ubuntu 21.04 尝试默认使用 Wayland

Ubuntu 17.10 中曾默认使用 Wayland，但在为 LTS 发布做准备时，开发团队对此方案并不自信，所以在 Ubuntu 18.04 LTS 中选择了继续默认使用 X.Org。

CERN 开放式硬件许可证获 OSI 批准

允许用户在得到许可的前提下，自由混合设计与专有设计，类似于 Apache 2.0。

Ubuntu 正在开发全新的安装程序

Ubuntu 目前正在使用的桌面端镜像文件采用的还是基于 Ubiquity 的安装程序，而 Ubiquity 最早是于 2010 年首次推出，在当时它具有不错的功能性和适度的快速性，随着时间的推移，Ubiquity 古老且复杂的代码库所带来的问题就暴露出来的。

Angular 发布新调试指南

CentOS 联合创始人成立公司赞助 Rocky Linux

既然 Rocky Linux 的目标是成为 CentOS 的完全替代品，将 CentOS 用户都吸引过来，如果没有充足的资金来源，实现目标的概率基本为零。

GCC 11 的静态分析功能将得到增强

CKEditor 5 v25.0.0 发布：改进图像管理、以及对大量文本的格式化

重新设计了注释功能的架构，从而实现将内联注释和边栏注释同时用于协作功能，此外还对块编辑功能进行了更改，因此用户可以更轻松地处理内容块。

Linux 或将限制模块调用某些寄存器

安全性的考虑。

开源基础设施基金会 2020 年度报告发布

FreeBSD 降低对 i386 架构的支持

i386 架构诞生已有 35 年，主要芯片公司基本上不再提供 i386 芯片，主流的 Linux 发行版也都陆续停止了对该架构的支持。

OpenSSF 推出工具和数据集 CVE 基准，建立安全评分卡

Linux 基金会与多家硬件和软件厂商合作，成立的开源安全基金会（OpenSSF），旨在提升开源软件安全性。

Facebook 2020 开源回顾

12w 次 commit、新增 200 个公开仓库。

VideoLAN 项目开源 20 周年

VideoLAN 发展成为一个法国的非营利组织，并在自由软件多媒体世界中开发了众多解决方案，包括 VLC 等。

微软组建 Rust 开发人员团队

将在内部系统上工作，同时与 Rust 开源社区合作。

VS Code Go 语言扩展将默认启用 gopls

gopls 提供了很多 IDE 的功能，例如自动补全、重构、工作空间符号搜索、代码美化和诊断分析任何兼容的编辑器前端。该语言服务器的推出是从早期命令行工具集合的一次根本转变。

开源项目应如何处理版权声明？

有一段时间，如果未在已出版的材料中包含版权声明，可能会导致完全丧失美国版权法提供的权利；当美国最终加入许多其他国家已经加入的"伯尔尼公约"时，情况发生了变化。

开源许可证的变更带给我们什么启示？

Elastic 公司与云服务厂商的“积怨”由来已久。

开源游戏引擎 Godot 4.0 完成了大量 CPU 和 GPU 优化

httpd 使用 Rust 重写 mod_ssl 模块

内存安全漏洞在安全领域一直都是严重的问题， httpd 也爆发过不少因使用 C 语言而引发的内存安全性问题。因此不难理解 ISRG 希望通过使用 Rust 重写 httpd 关键组件来提升安全性。

ECMAScript 2021 特性预览

String.prototype.replaceAll、Promise.any、WeakRefs、Numeric separators……

“工具人”赵生宇：清北本硕，为开源从阿里辞职去同济读博

开放式组织的数字化治理，用数字化工具让开源运营更简单。

Go 泛型提案有望被接受，已进入"Likely Accept"状态

进入此状态意味着提案审核小组已就接受提案达成了初步共识，下一步将基于此继续讨论以确定是否接受提案。
