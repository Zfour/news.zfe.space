зӣҙи§Ӯзҡ„ж„ҹеҸ—дёҖдёӢMacrotasksе’ҢMicrotasks

зңӢиҝҮдёҖзҜҮе…¬дј—еҸ·ж–Үз« дёӢйқўзҡ„з•ҷиЁҖ:

йӮЈдёӘжүҖи°“зҡ„mtaskе’Ңtaskзҡ„еҢәеҲ«жҲ‘е№¶дёҚи®ӨеҗҢ...пјҢжҲ‘и®ӨдёәдәӢд»¶еҜ№еҲ—еҸӘжңүдёҖдёӘпјҢе°ұжҳҜtaskгҖӮ

зү№еҲ«жҳҜеҜ№дәҺJSејӮжӯҘзј–зЁӢжҖқз»ҙиҝҳдёҚеӨӘзҶҹжӮүзҡ„еҗҢеӯҰпјҢжҜ”еҰӮдёӨе№ҙеүҚд»ҺjavaиҪ¬жҲҗjavascriptеҗҺзҡ„жҲ‘пјҢеҜ№дәҺиҝҷз§ҚејӮжӯҘзҡ„и°ғз”ЁйЎәеәҸе…¶е®һеҫҲйҡҫзҗҶи§ЈгҖӮ

дёҚиҝҮжңүдёҖдёӘзү№еҲ«иғҪиҜҙжҳҺMacrotasksе’ҢMicrotasksзҡ„дҫӢеӯҗ:

// жҷ®йҖҡзҡ„йҖ’еҪ’, йҖ жҲҗжӯ»еҫӘзҺҜ, йЎөйқўж— е“Қеә”
function callback() {
    console.log('callback');
    callback();
}
callback();

дёҠйқўзҡ„д»Јз ҒзӣёдҝЎеӨ§е®¶йқһеёёеҘҪзҗҶи§ЈпјҢдёҖдёӘеҫҲз®ҖеҚ•зҡ„йҖ’еҪ’пјҢз”ұдәҺдәӢд»¶еҫӘзҺҜеҫ—дёҚеҲ°йҮҠж”ҫпјҢUIжёІжҹ“ж— жі•иҝӣиЎҢеҜјиҮҙйЎөйқўж— е“Қеә”гҖӮ

йҖҡеёёжҲ‘д»¬еҸҜд»ҘдҪҝз”ЁsetTimeoutжқҘиҝӣиЎҢж”№йҖ пјҢжҲ‘д»¬жҠҠдёӢдёҖж¬Ўжү§иЎҢж”ҫеҲ°ејӮжӯҘйҳҹеҲ—йҮҢйқўпјҢдёҚдјҡжҢҒд№…зҡ„еҚ з”Ёи®Ўз®—иө„жәҗпјҢиҝҷе°ұжҳҜжҲ‘д»¬иҜҙзҡ„Macrotasks:

// MacrotasksпјҢдёҚдјҡйҖ жҲҗжӯ»еҫӘзҺҜ
function callback() {
  console.log('callback');
  setTimeout(callbackпјҢ0);
}


callback();

йҖҡиҝҮдёҠж–ҮжҲ‘д»¬д№ҹеҸҜд»ҘзҹҘйҒ“еҪ“ж¬ЎдәӢд»¶еҫӘзҺҜдә§з”ҹзҡ„ж–°Microtasksдјҡз«ӢеҚіжү§иЎҢпјҢеҗҢж—¶еҪ“ж¬ЎдәӢд»¶еҫӘзҺҜиҰҒзӯүеҲ°жүҖжңүMicrotasksйҳҹеҲ—жү§иЎҢе®ҢжҜ•еҗҺжүҚдјҡз»“жқҹгҖӮжүҖд»ҘеҪ“жҲ‘д»¬зҡ„MicrotasksеңЁдә§з”ҹж–°зҡ„д»»еҠЎзҡ„еҗҢж—¶пјҢдјҡеҜјиҮҙMicrotasksйҳҹеҲ—дёҖзӣҙжңүд»»еҠЎзӯүеҫ…жү§иЎҢпјҢиҝҷж¬ЎдәӢд»¶еҫӘзҺҜж°ёиҝңдёҚдјҡйҖҖеҮәпјҢд№ҹе°ұеҜјиҮҙдәҶжҲ‘д»¬зҡ„жӯ»еҫӘзҺҜгҖӮ

// MicrotasksпјҢеҗҢж ·дјҡйҖ жҲҗжӯ»еҫӘзҺҜпјҢйЎөйқўж— е“Қеә”
function callback() {
  console.log('callback');
  Promise.resolve().then(callback);
}

Microtasks дёҺ Promise A+

еҪ“з„¶пјҢдёҠж–Үи§ЈеҶідәҶжң¬дәәе…ідәҺMicrotasksзҡ„зӣёе…із–‘иҷ‘ (зү№еҲ«жҳҜжңүдәәжӢҝеҮәдёҖж®өеҸӮжқӮsetTimeoutе’ҢPromiseзҡ„д»Јз Ғи®©дҪ зңӢд»Јз Ғиҫ“еҮәйЎәеәҸж—¶) зҡ„еҗҢж—¶пјҢд№ҹи®©жҲ‘еӣһеҝҶиө·дјјд№ҺжӣҫеҮ дҪ•ж—¶д№ҹеңЁе“ӘйҮҢзңӢеҲ°иҝҮе…ідәҺMicrotaskзҡ„еӯ—зңјгҖӮ

з»ҸиҝҮеӨҡж—Ҙзҡ„еҜ»жүҫпјҢз»ҲдәҺеңЁд»ҘеүҚеҶҷиҝҮзҡ„дёҖзүҮе…ідәҺPromiseзҡ„жҖ»з»“ж–Үз«  жү“ејҖPromiseзҡ„жӯЈзЎ®е§ҝеҠҝ йҮҢжүҫеҲ°дәҶгҖӮиҜҘж–ҮйҖҡиҝҮдёҖдёӘе®һдҫӢиҜҙжҳҺдәҶж–°е»әPromiseзҡ„д»Јз ҒжҳҜдјҡз«ӢеҚіжү§иЎҢзҡ„пјҢе№¶дёҚдјҡж”ҫеҲ°ејӮжӯҘйҳҹеҲ—йҮҢ:

var d = new Date();


// еҲӣе»әдёҖдёӘpromiseе®һдҫӢпјҢиҜҘе®һдҫӢеңЁ2з§’еҗҺиҝӣе…ҘfulfilledзҠ¶жҖҒ
var promise1 = new Promise(function (resolveпјҢreject) {
  setTimeout(resolveпјҢ2000пјҢвҖҳresolve from promise 1вҖҷ);
});


// еҲӣе»әдёҖдёӘpromiseе®һдҫӢпјҢиҜҘе®һдҫӢеңЁ1з§’еҗҺиҝӣе…ҘfulfilledзҠ¶жҖҒ
var promise2 = new Promise(function (resolveпјҢreject) {
  setTimeout(resolveпјҢ1000пјҢpromise1); // resolve(promise1)
});


promise2.then(
  result => console.log(вҖҳresult:вҖҷпјҢresultпјҢnew Date() - d),
  error => console.log(вҖҳerror:вҖҷпјҢerror)
)

жҲ‘д»¬еҫ—еҲ°дёӨзӮ№з»“и®ә:

  • йӘҢиҜҒдәҶPromise/A+дёӯзҡ„2.3.2и§„иҢғ
  • ж–°е»әPromiseзҡ„д»Јз Ғж—¶дјҡз«ӢеҚіжү§иЎҢзҡ„ (иҝҗиЎҢж—¶й—ҙжҳҜ2з§’иҖҢдёҚжҳҜ3з§’)

дҪҶжҳҜеҪ“ж—¶жң¬дәәеҝҪз•ҘдәҶPromise/A+зҡ„зӣёе…іжіЁи§ЈеҶ…е®№:

Here вҖңplatform codeвҖқ means engineпјҢenvironmentпјҢand promise implementation code. In practiceпјҢthis requirement ensures that onFulfilled and onRejected execute asynchronouslyпјҢafter the event loop turn in which then is calledпјҢand with a fresh stack. This can be implemented with either a вҖңmacro-taskвҖқ mechanism such as setTimeout or setImmediateпјҢor with a вҖңmicro-taskвҖқ mechanism such as MutationObserver or process.nextTick. Since the promise implementation is considered platform codeпјҢit may itself contain a task-scheduling queue or вҖңtrampolineвҖқ in which the handlers are called.

жҳҜзҡ„пјҢиҝҷе°ұжҳҜжң¬дәәдёҺMicroTasksзҡ„з¬¬дёҖж¬ЎзӣёйҒҮпјҢжІЎжңүдёҖи§Ғй’ҹжғ…иҝҳзңҹжҳҜйқһеёёжҠұжӯүе•ҠгҖӮ

иҜҘжіЁи§ЈиҜҙжҳҺдәҶPromiseзҡ„ onFulfilled е’Ң onRejected еӣһи°ғзҡ„жү§иЎҢеҸӘиҰҒзЎ®дҝқжҳҜеңЁ thenиў«и°ғз”ЁеҗҺејӮжӯҘжү§иЎҢе°ұеҸҜд»ҘдәҶгҖӮе…·дҪ“е®һзҺ°жҲҗ setTimeout дјјзҡ„ macrotasks жңәеҲ¶жҲ–иҖ… process.nextTick дјјзҡ„microtasksжңәеҲ¶йғҪеҸҜд»ҘпјҢе…·дҪ“и§Ҷе№іеҸ°д»Јз ҒиҖҢе®ҡгҖӮ

дёәд»Җд№ҲйңҖиҰҒMicrotasks

жҗңзҙўеј•ж“ҺиғҪжүҫеҲ°зҡ„зӣёе…іж–Үз« еҹәжң¬йғҪжҢҮеҗ‘дәҶдёҖзҜҮгҖҠTasksпјҢmicrotasksпјҢqueues and schedulesгҖӢпјҢд№ҹи®ёиҝҷе°ұжҳҜдј иҜҙдёӯеҺҹзҪӘзҡ„еҸ‘жәҗд№Ӣең°еҗ§гҖӮ

Microtasks are usually scheduled for things that should happen straight after the currently executing scriptпјҢsuch as reacting to a batch of actionsпјҢor to make something async without taking the penalty of a whole new task.

з®ҖеҚ•жқҘиҜҙпјҢе°ұжҳҜеёҢжңӣеҜ№дёҖзі»еҲ—зҡ„д»»еҠЎеҒҡеҮәеӣһеә”жҲ–иҖ…жү§иЎҢејӮжӯҘж“ҚдҪңпјҢдҪҶжҳҜеҸҲдёҚжғійўқеӨ–д»ҳеҮәдёҖж•ҙдёӘејӮжӯҘд»»еҠЎзҡ„д»Јд»·гҖӮеңЁиҝҷз§Қжғ…еҶөдёӢпјҢMicrotasksе°ұеҸҜд»Ҙз”ЁжқҘи°ғеәҰиҝҷдәӣеә”еҪ“еңЁеҪ“еүҚжү§иЎҢи„ҡжң¬з»“жқҹеҗҺз«Ӣй©¬жү§иЎҢзҡ„д»»еҠЎгҖӮ

The microtask queue is processed after callbacks as long as no other JavaScript is mid-executionпјҢand at the end of each task. Any additional microtasks queued during microtasks are added to the end of the queue and also processed.

еҚ•зӢ¬зңӢMacrotasksе’Ң MicrotasksпјҢжү§иЎҢйЎәеәҸеҸҜд»ҘжҖ»з»“еҰӮдёӢ:

  • еҸ–еҮәMacrotasksд»»еҠЎйҳҹеҲ—зҡ„дёҖдёӘд»»еҠЎпјҢжү§иЎҢ;
  • еҸ–еҮәMicrotasksд»»еҠЎйҳҹеҲ—зҡ„жүҖжңүд»»еҠЎпјҢдҫқж¬Ўжү§иЎҢ;
  • жң¬ж¬ЎдәӢд»¶еҫӘзҺҜз»“жқҹпјҢзӯүеҫ…дёӢж¬ЎдәӢд»¶еҫӘзҺҜ;

д»ҺиҝҷдёӘж–№йқўжҲ‘д»¬д№ҹеҸҜд»ҘзҗҶи§Јдёәд»Җд№ҲPromise.thenиҰҒиў«е®һзҺ°жҲҗMicrotasksпјҢеӣһи°ғеңЁе®һзҺ°Promise/A+и§„иҢғ (еҝ…йЎ»жҳҜејӮжӯҘжү§иЎҢ)зҡ„еҹәзЎҖдёҠпјҢд№ҹдҝқиҜҒиғҪеӨҹжӣҙеҝ«зҡ„иў«жү§иЎҢпјҢиҖҢдёҚжҳҜи·ҹMacrotasksдёҖж ·еҝ…йЎ»зӯүеҲ°дёӢж¬ЎдәӢд»¶еҫӘзҺҜжүҚиғҪжү§иЎҢгҖӮеӨ§е®¶еҸҜд»ҘйҮҚж–°жү§иЎҢдёҖдёӢдёҠж–ҮеҜ№жҜ”Macrotasksе’ҢMicrotasksж—¶дёҫзҡ„дҫӢеӯҗпјҢд№ҹдјҡеҸ‘зҺ°д»–д»¬дёӨзҡ„еҚ•дҪҚж—¶й—ҙеҶ…зҡ„жү§иЎҢж¬Ўж•°жҳҜдёҚдёҖж ·зҡ„гҖӮ

еҸҜд»ҘиҜ•жғідёҖдәӣз»јеҗҲдәҶејӮжӯҘд»»еҠЎе’ҢеҗҢжӯҘд»»еҠЎзҡ„зҡ„Promiseе®һдҫӢпјҢMicrotasksеҸҜд»ҘдҝқиҜҒе®ғд»¬жӣҙеҝ«зҡ„еҫ—еҲ°жү§иЎҢиө„жәҗпјҢдҫӢеҰӮ:

new Promise((resolve) => {
  if(/* жЈҖжҹҘиө„жәҗжҳҜеҗҰйңҖиҰҒејӮжӯҘеҠ иҪҪ */) {
    return asyncAction().then(resolve);
  }
  // зӣҙжҺҘиҝ”еӣһеҠ иҪҪеҘҪзҡ„ејӮжӯҘиө„жәҗ
  return syncResource;
